# Amusement park ride eligiblity checker

## Summary
This C++ program asks the user for their age and height and decides if they are eligible for a ride based on the following rules:
- Age must be 13 or older
- Height must be 150 or taller 
it uses cin to get input from the user and if statements to check eligiblity. The program then prints a message tellling the user if they can ride or not.
